# Practical 04 – Multiple Linear Regression

Objective:
To implement multiple linear regression using the Titanic dataset.

Input Variables:
Age, Pclass, SibSp, Parch

Output Variable:
Fare

Steps:

2. Selected multiple input features
3. Split data into training and testing sets
4. Trained multiple linear regression model
5. Analyzed coefficients
6. Evaluated model using MSE and R² score

Tools:
- Python
- Pandas
- Scikit-learn
- Matplotlib
